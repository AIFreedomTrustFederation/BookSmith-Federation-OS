# Code of Conduct

BookSmith Federation OS exists to help people create, preserve, publish, and govern knowledge with dignity, sovereignty, and transparency.

## Expected behavior

Contributors are expected to:

- treat others with respect
- assume good faith when possible
- welcome beginners and experienced builders alike
- explain technical concerns clearly
- protect user agency, privacy, and authorship
- keep disagreements focused on the work

## Unacceptable behavior

Unacceptable behavior includes:

- harassment, threats, or personal attacks
- discriminatory language or conduct
- doxxing or exposing private information
- coercive behavior around identity, money, governance, or authorship
- intentionally hiding AI-generated changes or provenance
- attempting to overwrite user canon without approval

## Stewardship standard

This project should be governed by the principle:

```text
User input = accepted canon
AI output = proposed patch until approved
```

Technical power should serve human authors, organizations, trusts, and communities rather than replacing their agency.

## Reporting

If a conduct concern arises, open a private communication channel with the project steward or repository owner. When public GitHub issues are used, avoid exposing private personal information.
